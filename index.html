📊 유튜브 글로벌 트렌드 분석

⏳ 아직 첫 자동 수집 전이에요. 저장소의 Settings → Secrets and variables → Actions에 YT_API_KEY를 등록하고, Actions 탭에서 collect 워크플로를 한 번 실행하면 데이터가 나타나요. (이후엔 6시간마다 자동)

🔑 무료 API 키가 필요해요 (5분, 무료)

  1. console.cloud.google.com에 구글 계정으로 로그인
  2. 상단에서 프로젝트 만들기 (이름은 아무거나, 예: my-trend)
  3. ☰ 메뉴 → API 및 서비스 → 라이브러리 → "YouTube Data API v3" 검색 → 사용 설정
  4. ☰ 메뉴 → API 및 서비스 → 사용자 인증 정보 → + 사용자 인증 정보 만들기 → API 키
  5. 생성된 키(AIza로 시작)를 복사해서 아래에 붙여넣기
💡 키는 이 컴퓨터의 브라우저에만 저장되고 외부로 전송되지 않아요.
💡 키 만들 때 "키 제한" 설정은 하지 마세요 (제한 없음 상태여야 파일에서 작동).
💡 무료 한도는 하루 10,000포인트 — 이 사이트 기준 하루 30번 이상 새로고침 가능한 양이에요.
💡 결제수단(카드)을 등록하지 않는 한 과금은 원천적으로 불가능해요. 한도를 넘으면 요금이 나가는 게 아니라 그냥 멈추고, 이 사이트도 한도 직전에 자동으로 새로고침을 차단해요.
